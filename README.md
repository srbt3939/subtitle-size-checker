# web-subtitle-size-checker
字幕キャプションを付けたときのサイズ感を確かめるウェブページ

## 仕様　
640x360の映像に対して、MediaConvertで設定ができるあたかもウェブ上で確認できるサイト

## GitHub Pages サイト
このリポジトリの `index.html` を使って GitHub Pages に公開できます。
たとえば `https://<ユーザー名>.github.io/web-subtitle-size-checker/` のようなURLになります。
サイトでは、入力欄に字幕を入力すると擬似映像上に字幕が重ねられます。

## 使い方
1. 下の入力欄に字幕テキストを入力します。改行も反映されます。
2. 面積640×360ピクセルの黒いボックス上で、文字サイズや行数のイメージを確認します。
3. MediaConvertの設定値と照らし合わせてください。
